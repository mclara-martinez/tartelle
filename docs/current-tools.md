# Herramientas Actuales - Tartelle

## Herramientas en Uso

### Siigo (POS / Facturacion)
- **Uso:** Facturacion electronica, POS en punto de venta
- **Estado:** Funcional para facturacion, inventario no esta bien parametrizado
- **Problemas:**
  - Las recetas estan cargadas con gramos pero no descuenta inventario correctamente
  - Inventario no se actualiza en tiempo real
  - Admin tiene que subir produccion manualmente
- **Datos disponibles:** Estadisticas de ventas por semana/año (exportables)
- **Tiempo en uso:** ~1.5 años

### Excel / Google Sheets
- **Uso:** Registro de pedidos, seguimiento de entregas, produccion diaria
- **Funcionalidades actuales:**
  - Filtro por fecha de entrega
  - Precios automaticos por producto
  - Descuentos por cliente configurados
  - Suma automatica con domicilio
  - Chequeo de entrega (si/no)
  - Numero de factura electronica
- **Usuarios:** Admin principal

### WhatsApp (Grupos)
- **Grupo de ventas:** Pedidos de clientes
- **Grupo de Rappi:** Fotos y ultimos 4 digitos de pedidos Rappi
- **Grupo de produccion:** Lista de produccion del dia siguiente (se envia 6pm)
- **Grupo de inventario:** Reporte diario de producto terminado (noche)

### Rappi
- **Uso:** Canal de venta
- **Acceso:** Ninas de cocina manejan la plataforma directamente
- **Problemas:** No conectado con nada, sobreventa frecuente

### Instagram
- **Uso:** Canal de venta menor, redes sociales
- **Manejo:** Andrea directamente

### inDrive
- **Uso:** Domicilios ocasionales (complemento al domiciliario fijo)
